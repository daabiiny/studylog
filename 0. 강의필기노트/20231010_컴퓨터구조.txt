<컴퓨터 하드웨어>
1. CPU Central Processing Unit 중앙처리장치
 - 생각 처리
 - 컴퓨터 성능에 가장 중요
 - 클럭스피드 : CPU의 출력 수치
 - Hz(헤르츠) : 1과 0을 보내는 파장크기
 - 우리 컴퓨터 : 37GHz 초당 37번

2. RAM Random Access Memory 주기억장치
 - 당장 실행중인 프로그램 (Running Process)
 - RAM이 크면 컴퓨터 빠름

3. HDD Hard Disk Drive 보조기억장치
 - 당장은 아니지만 나중에 써먹어야 하는 정보
 - 비휘발성
 - 보관, 기록용
 - 속도 단위 : RPM Round Per Minute
 - 속도 굉장히 느림

* SSD
- 가격비쌈, 가성비떨어짐

<Network 네트워크> Win + R -> cmd -> ipconfig
 : 두대 이상의 컴퓨터와 컴퓨터가 서로 연결되어있는 상태에서 Data를 주고 받으면서 하는 작업 또는 그 환경 

1. IP Address Internet Protocol
 * Internet : 서로 다른 네트워크를 묶어서 만든 거대한 네트워크
    Protocol : 규격, 규칙과 약속
 : 인터넷을 하기 위해서 지켜야 할 규칙과 약속

- IP Address : 2진수 기반 32bit 형식으로 주소 구성 (IPv4)
- 클래스 : 
- IP 구성요소
①Network Bit 앞자리가 같으면 통신이 됨 (동네 주소)
②Host Bit (개인 주소)
ex) 192.168.10.1 (10진수)
.으로 구분된 파트는 255를 넘을 수 없음

2. Subnet Mask
 : 동네주소와 개인주소를 나누는 기준점 
ex) 255.255.255.0
        네트워크         호스트

3. Gateway
 : 같은 공간에서 다른 곳으로 가기 위해서 첫번째로 빠져나가는 문
   같은 네트워크 안에 있어야함
   다른 사이트 (유투브, 네이버 등)로 가기위해서 거쳐야함
ex) 192.168.112.1

4. DNS Server (like 전화번호부)
: 글자에서 숫자로 바꿔주는 것
  도메인 이름을 IP 주소로 변환하는 것
