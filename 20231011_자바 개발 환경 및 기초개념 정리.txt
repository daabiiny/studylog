< 오늘 알아야 하는 용어 >

JRE (Java Runtime Environmont, 자바 실행 환경)
: 자바 가상머신을 포함한다. 자바로 작성된 프로그램을 실행하기 위해서 필요하다

JDK (Java Development Kit, 자바 개발 도구)
: 자바를 이용하여 프로그램을 개발할 수 있다. 자바 컴파일러가 포함되어 있다 

source 혹은 src, *.java, 소스 파일 
: 사람이 자바 문법으로 작성한 프로그램 소스 코드, 컴파일하면 바이트코드로 변환된다

byte code, bin, *.class, 바이트 코드 파일 
: 컴파일러를 거쳐 소스코드가 변환된 형태, JVM위에서 실행할 수 있다

JVM (Java Virtual Machine) 자바 가상 머신
: 자바 프로그램을 실행하기 위해 운영체제 위에서 실행중인 프로그램

IDE (Intergrated Development Environmen) 통합 개발 환경
: 소스작성, 파일관리, 컴파일 및 인터프리트, 자동완성, 디버깅, 실행을 하나의 프로그램에서 지원한다

< 자바의 장점 >
1) 포인터를 사용하지 않기 때문에 메모리 주소 관리 및 위험성이 낮아진다
2) 가비지 컬렉터에 의해서 사용되지 않는 데이터를 주기적으로 자동으로 소거시키기 때문에 공간 관리가 편리하다
3) 자료형을 명시하기 때문에 불필요한 추론이 적어진다. 명확성을 강조한다. ( 대신 문법은 길어짐 ) 자바 수다쟁이 삉

1970년대 C (데니스 리치)
 - 서로 다른 기종의 컴퓨터에서도 같은 언어로 프로그래밍이 가능하도록
1980년대 C++ (비야네 스트로스트룹)
 - C언어의 기본을 유지하면서 객체지향의 개념을 도입

 - 메모리 관리, 포인터 삭제, 서로 다른 HW/SW에서 똑같이 실행되는 프로그램
1990년대 Python (귀도 반 로섬)

JRE (Java Runtime Environmont, 자바 실행 환경)
 - 자바로 개발된 프로그램을 실행하기 위해 필요한 요소
 - 마인크래프트를 실행하기 위해서는 자바가 설치되어 있어야 한다 (이때 자바는 JRE를 말함)

JDK (Java Development Kit, 자바 개발 도구)
 - 자바 언어로 프로그램을 개발하기 위한 도구 모음, 실행 및 테스트를 위해 JRE를 포함한다
 - 자바 기반의 안드로이드를 앱을 개발하기 위해서 자바를 설치해야 한다 (이때 자바는 JDK를 말함)
